About

Research

IDEA Framework: Integrated Diplomatic Enterprise Architecture Design

Master's Thesis • 2024-2025

Developing a novel framework that integrates computational diplomacy principles with enterprise architecture stakeholder communication. The framework uses diplomatic action primitives (PROPOSE, CLARIFY, CONSTRAIN, REVISE, AGREE, ESCALATE, DEFER) to structure and improve communication patterns in enterprise architecture decision-making.

Key Contributions:

  • Framework design integrating diplomatic theory with TOGAF and Zachman frameworks
  • Evaluation methodology using Architecture Decision Records (ADRs) as test cases
  • Comparative analysis against baseline models (GPT-4, Claude Sonnet)
  • Implementation using transparent models (OLMo-7B) for reproducibility

Methods: NLP, Task-Oriented Dialogue Systems, Enterprise Architecture, Computational Social Science
